fetchRecentPosts

Fetches recent posts from the posts collection ordered by date created.

fetchRecentPosts(limit: number): Promise<Array<Object>>
Parameters
limit (number = 10) The maximum number of recent posts to fetch (default is 10).
Returns
Promise<Array<Object>>: A promise that resolves to an array of recent posts.
Throws
  • any: Will throw an error if there is an issue fetching recent posts.

followingPosts

Endpoint to fetch posts for users or categories that a user is following.

followingPosts(req: Object, res: Object): Object
Parameters
req (Object) Express request object.
Name Description
req.query Object The query parameters.
req.query.followingList string JSON stringified array of followed user/category objects.
req.query.category string The category to filter posts by.
req.query.userFavCategories string JSON stringified array of user's favorite categories and sub-categories.
res (Object) Express response object.
Returns
Object: 200 - An array of arrays, each containing posts.
Returns
Object: 400 - Missing parameters.
Returns
Object: 500 - Internal Server Error.
